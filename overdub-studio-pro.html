Overdub Studio

Sprechen, mischen, fertig — alles in einer App. Vom Skript bis zum sendefertigen Mix gegen Bewegtbild: Video laden → Skript laden → Sprechen → fertig.

Highlights

Eine Sache richtig

Pro Tools, Reaper und Studio One können alles — aber nichts davon ist auf den video-synchronen Voice-Over-Workflow zugeschnitten. Overdub Studio schon. Vier Top-Features übernehmen das Handwerk, damit du dich aufs Sprechen konzentrieren kannst.

Auto Edit

Schnitt ohne Schnitt-Handwerk

Right-Click → „Auto-Cut" trennt Takes an Stille-Passagen. Strip Silence per One-Click. Drag-and-Drop Take-Editing auf der Timeline, Punch-In/Out, In-Take-Range-Trim, Multi-Take-Layering — mit Undo/Redo über alle Operationen.

Auto Audio

Voice-Channel-Strip vollautomatisch

VoiceAnalyzer erstellt aus jedem Take ein Profil. SmartDenoiser, AutoEQ (5-Band + HP80), AutoComp und SmartLeveler (bis +24 dB Rescue) konfigurieren sich selbst. Sendefertig ohne ein einziges Reglerklicken.

AutoMix

Ein Klick, sendefertig auf R128

SmartDucker (Mehrband, Look-ahead, S-Curve), LRA-Korrektur ≤ 6 LU, 4× oversampled True-Peak-Limiter. Single MIXTYPE-Slider: SPORT → MAGAZIN → NARRATION. Ein Regler, drei Mixe — alles auf -23 LUFS / -1 dBTP.

Smart Prompter

Prompter, der dir folgt

Apple SFSpeechRecognizer (on-device, DE+EN) verfolgt deine Stimme live. ScriptAligner mit Rolling-Window Fuzzy-Match. Lese-Position bleibt fix, Text fließt durch — wie ein Studio-Prompter mit Strahlteiler. Kein Pedal, kein Slider.

Video

MXF Round-Trip + AVPlayer

Native AVPlayer-Wiedergabe für alle macOS-Codecs. MXF mit 8 Mono-Tracks (Sum + M+E + OT) — Layout bleibt beim Export erhalten. ffmpeg-Proxy für unkooperative Container, Sync-Genauigkeit ≤ 250 ms, Timecode-Burn-in.

Recording

Multi-Track mit Pre-Roll & LUFS

Voice 1 + 2 unabhängig, ARM/MUTE/SOLO pro Spur. 4-Sek-Pre-Roll mit Wood-Block-Click. Lock-free Audio-Thread, K-weighted LUFS-Meter (M+S+I), Broadcast-Meter im Studio-Look. Hardware-Monitoring für Zero-Latency.

Funktionsweise

Vier Schritte, eine App

Klassischer Workflow: vier Tools, drei Round-Trips, eine Stunde Setup. Hier: vier Schritte, eine App, fertig.

01

Video laden

Drag-and-Drop in den Player. MP4, MOV, MXF (XDCAM, IMX), WebM. Audio aus dem Video wird automatisch als Bett-Spur angelegt.

02

Skript laden

.txt, .rtf oder Cmd+V aus Word, Browser, Mail. Smart Prompter zeigt das Skript broadcast-typisch — feste Lese-Position, fließender Text.

03

Sprechen

REC drücken, 4-Sek-Pre-Roll, los. Der Prompter folgt deiner Stimme. Multi-Take, Auto-Cut an Stillen, beliebig oft wiederholen.

04

Mixen & Exportieren

MIXTYPE-Slider justieren, AutoMix klickt einmal — fertig auf -23 LUFS. Export als WAV, separater Voice-Stem, Video-Container mit eingebettetem Mix.

Technische Details

Built for Production

Native C++, JUCE 8, lokal — kein Cloud-Sync, kein Streaming.

Standards & DSP

  • Loudness ITU-R BS.1770-4 · EBU R128
  • Target -23 LUFS · -1 dBTP
  • True Peak 4× polyphase FIR
  • LRA-Ziel ≤ 6 LU (TV-Standard)
  • Sample-Rates 44.1 / 48 / 96 kHz
  • Latenz < 256 Samples Round-Trip

Audio Engine

  • Profil VoiceAnalyzer
  • Denoiser SmartDenoiser (FFT)
  • Channel Strip AutoEQ + AutoComp + SmartLeveler
  • Ducker SmartDucker (Mehrband, Look-ahead)
  • Prompter SFSpeechRecognizer DE/EN
  • Saturation Tube (optional)

Plattform & I/O

  • OS macOS 12+ Universal Binary
  • Architektur Apple Silicon + Intel x86_64
  • Video MP4, MOV, MXF, WebM
  • MXF Round-Trip Sum + M+E + OT
  • Interfaces alle macOS (RME, UA, MOTU, …)
  • Export WAV (16/24/32 bit) · Stems · Video-Mux
Was es nicht ist

Keine DAW.
Kein Video-Editor.
Eine Sache richtig.

Overdub Studio ist keine universelle DAW (kein MIDI, keine Plug-In-Slots) und kein Video-Editor. Es ist ein Werkzeug für genau einen Workflow — Voice-Over gegen Bewegtbild — und macht den richtig. Eine Lizenz, kein Abo, keine Cloud, kein Streaming. Broadcast-Material verlässt nicht das Studio.
